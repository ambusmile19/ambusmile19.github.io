Fall Detection on Elderly People

An interdisciplinary research project using graphical clustering techniques.

  • Developed a motion-tracking pipeline for monitoring elderly movements.
  • Used Graph Neural Networks and clustering algorithms to detect falls.
  • Model identifies abnormal motion patterns with high sensitivity.
  • Useful for healthcare monitoring and smart home safety systems.

Connection to SIGNLIPDETECTRON

The computer vision framework used in SIGNLIPDETECTRON helped in developing a robust motion-detection system for this fall-detection project.
